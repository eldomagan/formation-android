Android

Android est une plateforme open source pour mobile qui fournit un environnement d'exécution pour des applications.

La plateforme android est constitué des éléments suivants:

  • Un système d'exploitation
  • Une suite de librairies de base (sqlite, OpenGL, WebKit..)
  • Un framework d'application
  • Un runtime
  • Des applis des base
  • Android SDK

Historique

  • 2008 Android version 1.0
  • 2009 v1.1 was released, v1.5 (CupCake), v1.6 (Donut), v2.0/2.1 (Eclair)
  • 2010 v2.2 (Froyo), v2.3 (GingerBread)
  • 2011 v3.0 (HoneyComb), v4.0 (Ice Cream Sandwich)
  • 2012 v4.1 (Jelly Bean)
  • 2013 v4.4 - 4.4.4, v4.4W - 4.4W.2 (KitKat)
  • 2014 v5.0 - 5.1.1 (Lollipop)
  • 2015 v6.0 - 6.0.1 (Marshmallow)
  • 2016 v7.0 - 7.1.1 (Nougat)

  • Les Activités

Les Activités

Chaque écran d'une application est gérer par une Activité.

C'est donc des Activités que nous allons utiliser pour interagir avec l'utilisateur

Une activité est le plus souvent composé de plusieurs widgets (View)

Les Services

Lorque nous avons besoin d'éxécuter certaines opérations qui n'ont pas besoin d'interagir avec l'utilisateur (téléchargement..). Les sevices sont exécutés en arrière plan.

Les Broadcast Receiver

Les Broadcast Receiver permettent à une application android de repondre à certains événements déclenchés par le System Android (Réception d'SMS, Appel Entrant...)

Hello Android !

Création du projet

  • Application Name: Nom de votre application
  • Company Domain: nom de domaine du développeur. Pas très important, on peut mettre n'importe quoi
  • Package Name: c'est composé du nom de domaine et du nom de l'application. Il est utilisé comme identifiant unique de l'application.
  • Project Location: Lieu où le repertoire du projet sera créé

Pour des éléments beaucoup plus complexe, le ArrayAdapter n'est plus adapté.

Il faut alors créer son adapter soi même


                            class User {
                                private String nom;
                                private String prenom
                                private String description;    

                                public User(String nom, String prenom, String description) {
                                    this.nom = nom;
                                    this.prenom = prenom;
                                    this.description = description;
                                }

                                // generate getters and setters
                            }
                        

                            <RelativeLayout xmlns:android="http://schemas.android.com/apk/res/android"
                                android:layout_width="match_parent"
                                android:layout_height="match_parent" >
                                <TextView
                                android:id="@+id/tvPrenom"
                                android:layout_width="wrap_content"
                                android:layout_height="wrap_content"
                                android:text="Prenom" />

                                <TextView
                                    android:id="@+id/tvNom"
                                    android:layout_width="wrap_content"
                                    android:layout_height="wrap_content"
                                    android:layout_toRightOf="@id/tvPrenom"
                                    android:text="Nom" />

                                <TextView
                                    android:id="@+id/tvDescription"
                                    android:layout_width="wrap_content"
                                    android:layout_height="wrap_content"
                                    android:layout_below="@id/tvPrenom"
                                    android:text="Description" />
                            </LinearLayout>
                        

                            public class UsersAdapter extends ArrayAdapter<User> {

                                public UsersAdapter(Context context, ArrayList<User> users) {
                                    super(context, 0, users);
                                }

                                @Override
                                public View getView(int position, View convertView, ViewGroup parent) {
                                
                                    User user = getItem(position);    
                                    
                                    if (convertView == null) {
                                        convertView = LayoutInflater.from(getContext()).inflate(R.layout.item_user, parent, false);
                                    }
                                
                                    TextView tvPrenom = (TextView) convertView.findViewById(R.id.tvPrenom);
                                    TextView tvNom = (TextView) convertView.findViewById(R.id.tvNom);
                                    TextView tvDescription = (TextView) convertView.findViewById(R.id.tvDescription);
                                
                                    tvPrenom.setText(user.getPrenom());
                                    tvNom.setText(user.getNom());
                                    tvDescription.setText(user.getDescription());
                                
                                    return convertView;
                                }
                            }
                        

                            ListView userList = (ListView) findViewById(R.id.user_list);
